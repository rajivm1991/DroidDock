DroidDock v0.3.0

Release Notes • February 13, 2026

What's New in v0.3.0

Skip Duplicate Downloads

Added a new setting to skip files that already exist in the destination folder. This avoids duplicate transfers and speeds up repeated download workflows.

Clearer Download Feedback

Download summaries now distinguish downloaded files, skipped duplicates, and failed items so transfer outcomes are immediately clear.

Homebrew Cask Install Path

Added a Homebrew Cask installation option in project documentation and website download sections to simplify setup and upgrades.
